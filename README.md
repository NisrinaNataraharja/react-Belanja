## üíª How To Install?

- Clone This Repo

```
git clone https://github.com/NisrinaNataraharja/react-Belanja.git
```

- Go To Folder Repo

```
cd ..
```

- Install Module

```
npm install
```

- Type ` npm start` To Start Website
